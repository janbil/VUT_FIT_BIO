School project for Biometric systems in academical year 2022/23

Project aims for implementation of tool for recognition of credibility of fingerprint.

This project includes:
	core_funcs.py - source file including some core functions
	gabor.py - source file implementing gabor filter
	estimation.py - main source file
	dokumentace.pdf - documantation of the project with key ideas
	README.txt - this file

Usage: estimation.py [options] sourceimage [MODE]

Options:
  -h, --help  show this help message and exit
  -i          Show intermediate images.

MODE sets filtering for edge recognition for given dataset. Default is BERGDATA
